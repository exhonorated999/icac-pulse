# Assets Directory

This directory contains image assets used throughout the application.

## Required Images

### waiting-esp-icon.png
- **Used in:** Dashboard - "Waiting on E.S.P." status card
- **Size:** Recommended 48x48px or similar (will be displayed at 48x48px)
- **Format:** PNG with transparency
- **Description:** Clock/timer icon with yellow/gold coloring indicating warrant waiting status

To add the image:
1. Save the ESP waiting icon as `waiting-esp-icon.png`
2. Place it in this directory
3. The Dashboard will automatically display it in the "Waiting on E.S.P." card
